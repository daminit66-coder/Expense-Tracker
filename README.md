# Expense Tracker

A command-line Expense Tracker built using Python and PostgreSQL.

## Features

- Add Expense
- View All Expenses
- Search by Product
- View Individual Columns
- Update Expense
- Delete Expense
- Maximum Price
- Minimum Price
- Total Spending
- Average Spending
- Category-wise Spending
- Top 5 Expensive Purchases

## Technologies

- Python
- PostgreSQL
- psycopg2
